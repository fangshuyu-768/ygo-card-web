# 游戏王卡牌介绍网页版

一个不依赖 Xcode 的静态网页原型，可以直接在浏览器里试用。

## 打开方式

直接双击 `index.html`，或用浏览器打开它。

## 当前功能

- 卡牌图片展示
- 按名称或作用搜索
- 搜索时自动补充线上卡牌库结果，并缓存查询结果
- 搜索结果会分批显示，向下浏览时自动继续加载更多
- 按怪兽、魔法、陷阱分类
- 收藏卡牌
- 随机查看一张卡
- 中文说明优先显示，并保留英文原文对照
- 朗读卡牌名称和中文说明
- 浏览器支持时可用内置语音识别按钮
- 在线卡牌会补充更中性的效果要点说明
- 每张卡会展示推荐用法，朗读时也会一起读出
- 详情页会补充使用定位标签和常见配合方向
- 启动时从仓库内的本地 JSON 读取大批量卡牌，不再依赖在线动态搜索

## 说明

- 当前仓库内包含一份从 `YGOPRODeck API` 拉取并裁剪后的本地卡牌数据
- 页面会优先加载本地卡牌库；如果本地文件读取失败，才回退到 8 张内置示例卡牌
- 图片目前仍使用线上链接，当前只把卡牌文本数据静态化到仓库
- 也可以直接用系统键盘或系统听写功能做“语音输入”
