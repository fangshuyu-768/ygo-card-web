# 游戏王卡牌介绍网页版

一个不依赖 Xcode 的静态网页原型，可以直接在浏览器里试用。

## 打开方式

直接双击 `index.html`，或用浏览器打开它。

## 当前功能

- 卡牌图片展示
- 按名称或作用搜索
- 搜索时自动补充线上卡牌库结果，并缓存查询结果
- 搜索结果会分批显示，向下浏览时自动继续加载更多
- 按怪兽、魔法、陷阱分类
- 收藏卡牌
- 随机查看一张卡
- 朗读卡牌名称和介绍
- 浏览器支持时可用内置语音识别按钮
- 在线卡牌会补充更中性的效果要点说明

## 说明

- 第一版内置 8 张示例卡牌
- 搜索框输入 2 个以上字符后，会自动从 `YGOPRODeck API` 补充更多卡牌
- 查询结果会保存在浏览器本地缓存里，减少重复请求
- 图片使用线上链接，当前仍属于原型方案
- 也可以直接用系统键盘或系统听写功能做“语音输入”
